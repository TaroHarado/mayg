# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release of mayhem.tax SDK
- Support for launching tax tokens on pump.fun
- Integration with mayhem protocol
- Support for 2022 token standard
- Configurable buy/sell tax rates (0-25%)
- Random tax variation support
- Devnet and mainnet support
- TypeScript type definitions
- Comprehensive documentation
- Example code snippets

### Features
- `MayhemTaxSDK` - Main SDK class
- `TokenLauncher` - Token deployment functionality
- `MayhemProgram` - On-chain program interaction
- Utility functions for address validation and formatting
