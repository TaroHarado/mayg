# mayhem.tax - Tax Token Platform

Платформа для создания tax токенов (2022 стандарт) на pump.fun через mayhem протокол.

## О проекте

mayhem.tax - это слой поверх протокола mayhem, который позволяет создавать tax токены используя стандарт 2022 на pump.fun. Протокол mayhem от pump.fun позволит делать рандомные транзакции, устанавливать комиссии на покупку/продажу и другие продвинутые функции.

**Запуск платформы произойдет после официального развертывания контракта mayhem со стороны pump.fun.**

## Особенности

- ✅ Чистый HTML/CSS/JS - никаких зависимостей от Next.js
- ✅ Tailwind CSS через CDN - современный дизайн
- ✅ Подключение Phantom кошелька
- ✅ Создание tax токенов с настраиваемыми комиссиями
- ✅ Поддержка 2022 стандарта
- ✅ Работает локально без проблем
- ✅ Все пути относительные

## Быстрый старт

### Вариант 1: Использовать готовый сервер (рекомендуется)
Просто запустите `start-server.bat` и откройте в браузере:
```
http://localhost:8080/
```

### Вариант 2: Использовать Node.js напрямую
```bash
node server.js
```

### Вариант 3: Использовать Python (если установлен)
```bash
python -m http.server 8080
```

## Структура проекта

- `index.html` - Главная страница со списком токенов
- `create.html` - Страница запуска нового токена
- `profile.html` - Страница профиля пользователя
- `faq.html` - Часто задаваемые вопросы
- `wallet.js` - Логика подключения Phantom кошелька
- `Vector.webp` - Логотип
- `favicon.ico` - Иконка сайта
- `server.js` - Простой HTTP сервер на Node.js
- `start-server.bat` - Скрипт для быстрого запуска

## Редактирование

Все файлы - это простой HTML с встроенным JavaScript. Вы можете легко:
- Изменить дизайн через Tailwind CSS классы
- Добавить функциональность в JavaScript
- Изменить тексты и контент напрямую в HTML

## Примечания

- Сайт требует HTTP сервер (не работает при открытии через file://)
- Используется Tailwind CSS через CDN (требуется интернет)
- Все пути относительные - работает везде
- Для работы с кошельком требуется установленный Phantom

## Остановка сервера

Нажмите `Ctrl+C` в окне терминала, где запущен сервер.
